Tutorial de Configuração do Ambiente PHP Local

Introdução

PHP é uma linguagem de script de código aberto muito popular, que é especialmente adequada para desenvolvimento web e pode ser incorporada em HTML. É usado por milhões de sites em todo o mundo devido à sua flexibilidade e simplicidade.

Configurando com PHP do site oficial

  1. Baixe a versão mais recente do PHP compatível com seu sistema operacional no site oficial do PHP.




  2. Após o download, extraia o arquivo para um diretório de sua escolha.




  3. Abra o prompt de comando e navegue até o diretório onde o PHP foi extraído.



  4. Inicie o servidor embutido com o comando: php -S localhost:80 -t "pastadetrabalho"



  5. Crie um arquivo PHP chamado exemplo.php e coloque algum código nele.


  6. Salve o arquivo na pasta de trabalho definida anteriormente.


  7. Abra o navegador e digite localhost/exemplo.php na barra de endereços para ver o resultado.

Configurando com XAMPP

  1. Baixe e instale o XAMPP.


  2. Após a instalação, abra o painel de controle do XAMPP e inicie os módulos Apache e MySQL.



  3. Coloque suas pastas de projetos e códigos PHP dentro da pasta htdocs, que está no diretório onde o XAMPPestá instalado.


  4. Crie um arquivo PHP chamado index.php dentro da pasta htdocs e adicione algum código a ele.


  5. Abra o navegador e digite localhost/pasta/exemplo.php na barra de endereços para ver o resultado.
