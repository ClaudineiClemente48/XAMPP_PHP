Trabalhando com PHP no XAMPP usando o PowerShell

  1. Faça o download do XAMPP: Visite o site oficial e baixe a versão compatível com o seu sistema operacional.






  2. Crie uma pasta para extrair os arquivos: Após o download, crie uma nova pasta e extraia os arquivos do XAMPP nela.




  3. Abra o PowerShell: No menu Iniciar, procure por "PowerShell" e abra o aplicativo.






  4. Crie um arquivo PHP: No PowerShell, navegue até o diretório onde o XAMPP foi extraído e execute o comando para criar um arquivo PHP.




  5. Edite o arquivo PHP: Abra o arquivo PHP com um editor de texto e adicione seu código PHP.




  6. Crie um arquivo HTML: Use o Bloco de Notas para criar um arquivo HTML com instruções simples para uma pessoa leiga.
